# TrafficSense

TrafficSense is a CNN-based Deep Learning project for Vehicle and Non-Vehicle Classification.

## Features
- Vehicle Detection using CNN
- Binary Image Classification
- Training vs Validation Accuracy Analysis
- Overfitting Study
- Prediction on New Images

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib

## Dataset
Vehicle Detection Image Set (Kaggle)
